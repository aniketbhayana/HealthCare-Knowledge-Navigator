# Healthcare Knowledge Navigator

A Retrieval Augmented Generation (RAG) application that enables evidence based querying of healthcare guidelines.

## Features

* Multi-document guideline search
* ChromaDB vector database
* SentenceTransformer embeddings
* Gemini 2.5 Flash powered responses
* Source citations
* Streamlit web interface

## Knowledge Base

* ADA Standards of Care 2026
* WHO Diabetes Guidelines
* NICE Diabetes Guidelines

## Tech Stack

* Python
* Streamlit
* ChromaDB
* Sentence Transformers
* Gemini 2.5 Flash

## Run Locally

pip install -r requirements.txt

streamlit run app.py
